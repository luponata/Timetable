# Timetable's Legend

1st argument = Year (4 digits)\
2nd argument = Month (Without "0" prefix)\
Optional "weekends" argument" = Show weekends in table

#### Python examples
```
./timemap.py 2022 6
./timemap.py 2022 11 weekends
```
#### Windows examples
```
./timemap_win.exe 2022 6
./timemap_win.exe 2022 11 weekends
```
